Partial Orders

Summary

A partial order is a relation on a set that is reflexive, antisymmetric, and transitive. Part of Functions and Relations.

  • Posets and Hasse diagrams
    • A poset (partially ordered set) is a pair $(P, \leq)$ where $\leq$ is a partial order on $P$.
    • A Hasse diagram draws the cover relation of a poset, omitting reflexive loops and edges implied by transitivity.
  • Total orders and lattices
    • A total order (or linear order) additionally requires that $a \leq b$ or $b \leq a$ for all $a, b \in P$.
    • A lattice is a poset in which every pair of elements has a least upper bound (the join) and a greatest lower bound (the meet).
  • Chains and antichains
    • A chain is a totally ordered subset of a poset.
    • An antichain is a set of pairwise incomparable elements.

Dilworth's theorem: in any finite poset, the minimum number of chains needed to cover $P$ equals the maximum size of an antichain.
