Advanced Combinatorics

Summary

Advanced combinatorics covers special enumerative sequences and group-theoretic counting techniques that arise in discrete mathematics. Part of Combinatorics.

  • Catalan numbers
    • The Catalan number $C_n = \dfrac{1}{n+1}\binom{2n}{n}$ counts balanced parenthesizations, binary trees with $n$ internal nodes, monotone lattice paths, and polygon triangulations.
  • Stirling numbers
    • The Stirling number of the first kind $\left[{n \atop k}\right]$ counts permutations of $[n]$ with exactly $k$ cycles.
    • The Stirling number of the second kind $\left{{n \atop k}\right}$ counts partitions of $[n]$ into $k$ nonempty blocks.
  • Bell numbers
    • The Bell number $B_n = \displaystyle\sum_{k=0}^{n} \left{{n \atop k}\right}$ counts all set partitions of $[n]$.
  • Integer partitions
    • An integer partition of $n$ is an unordered sum of positive integers equaling $n$; the generating function is $\prod_{k \geq 1} \dfrac{1}{1 - x^k}$.
  • Burnside's lemma and Pólya enumeration
    • Burnside's lemma counts the number of orbits of a finite group $G$ acting on a set $X$ as $\dfrac{1}{|G|}\displaystyle\sum_{g \in G} |X^g|$, where $X^g$ denotes the set of elements fixed by $g$.
    • Pólya's enumeration theorem refines this with cycle index polynomials for counting colorings up to symmetry.
