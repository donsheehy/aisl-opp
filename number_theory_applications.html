Applications of Number Theory

Summary

Number-theoretic applications are computer-science constructions built on divisibility, primes, and modular_arithmetic. Part of Number Theory.

  • RSA cryptography
    • Keys are built from two large primes $p, q$ with modulus $n = pq$.
    • Encryption computes $c \equiv m^e \pmod{n}$; decryption recovers $m \equiv c^d \pmod{n}$ with $ed \equiv 1 \pmod{\varphi(n)}$.
    • Security relies on the apparent difficulty of factoring $n$.
  • Hashing
    • Simple hash functions use $h(x) = x \bmod p$ for a prime $p$.
    • Universal hashing exploits modular arithmetic to give provable collision bounds.
  • Pseudorandom number generation
    • Linear congruential generators use the recurrence $x_{n+1} = (a x_n + c) \bmod m$.
    • The quality of the generated sequence depends on the choice of parameters $a$, $c$, and $m$.
