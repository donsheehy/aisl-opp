Random Variables

Summary

A random variable is a function $X : \Omega \to \mathbb{R}$ on a probability space that assigns a numerical value to each outcome. Part of Discrete Probability.

  • Discrete distributions
    • A Bernoulli$(p)$ random variable satisfies $\Pr[X=1] = p$ and $\Pr[X=0] = 1-p$.
    • A binomial$(n, p)$ random variable satisfies $\Pr[X=k] = \binom{n}{k} p^k (1-p)^{n-k}$.
    • A geometric$(p)$ random variable satisfies $\Pr[X=k] = (1-p)^{k-1} p$ for $k \geq 1$.
    • A Poisson$(\lambda)$ random variable satisfies $\Pr[X=k] = \dfrac{\lambda^k e^{-\lambda}}{k!}$.
  • Probability mass functions
    • The probability mass function (PMF) of $X$ is $p_X(x) = \Pr[X = x]$, with $\sum_x p_X(x) = 1$.
  • Cumulative distribution functions
    • The cumulative distribution function (CDF) of $X$ is $F_X(x) = \Pr[X \leq x]$, which is non-decreasing and right-continuous.
