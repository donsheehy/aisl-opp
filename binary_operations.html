Binary Operations

Summary

A binary operation on a set $A$ is a function $\ast : A \times A \to A$ that combines two elements to produce a third element of $A$. Part of Algebraic Structures.

  • Closure
    • Closure holds automatically when $\ast$ is defined as a function into $A$.
  • Associativity
    • $\ast$ is associative if $(a \ast b) \ast c = a \ast (b \ast c)$ for all $a, b, c \in A$.
  • Commutativity
    • $\ast$ is commutative if $a \ast b = b \ast a$ for all $a, b \in A$.
  • Identity element
    • An identity element $e$ satisfies $e \ast a = a \ast e = a$ for all $a \in A$.
  • Inverses
    • An inverse of $a$ is an element $a^{-1}$ satisfying $a \ast a^{-1} = a^{-1} \ast a = e$.

These properties give rise to monoids, groups, rings, and other algebraic structures.
