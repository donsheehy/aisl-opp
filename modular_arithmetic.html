Modular Arithmetic

Summary

Modular arithmetic is arithmetic in the quotient $\mathbb{Z}/n\mathbb{Z}$ obtained by identifying integers that are congruent modulo $n$. Part of Number Theory.

  • Congruences and residue classes
    • We say $a$ is congruent to $b$ modulo $n$, written $a \equiv b \pmod{n}$, if $n \mid (a - b)$.
    • Congruence modulo $n$ is an equivalence relation; its equivalence classes are called residue classes mod $n$.
  • Modular inverses
    • An integer $a$ has a multiplicative inverse mod $n$ if and only if the greatest common divisor satisfies $\gcd(a, n) = 1$.
    • The inverse can be computed via the extended Euclidean algorithm (see gcd).

Chinese Remainder Theorem: for pairwise coprime moduli $n_1, \ldots, n_k$, the system $x \equiv a_i \pmod{n_i}$ has a unique solution modulo $n_1 n_2 \cdots n_k$.

Fermat's Little Theorem: if $p$ is prime and $p \nmid a$, then $a^{p-1} \equiv 1 \pmod{p}$.

Euler's Theorem: if $\gcd(a, n) = 1$, then $a^{\varphi(n)} \equiv 1 \pmod{n}$, where $\varphi(n)$ denotes Euler's totient function, the count of integers in ${1, \ldots, n}$ coprime to $n$.
