Special Graph Classes

Summary

Special graph classes are families of graphs with additional structural properties, such as bipartiteness or planarity. Part of Graph Theory.

  • Bipartite graphs
    • A graph is bipartite if its vertex set partitions into two parts $X$ and $Y$ with all edges running between them; equivalently, a graph is bipartite if and only if it contains no odd cycle.
    • König's theorem states that in a bipartite graph, the maximum matching size equals the minimum vertex cover size.
  • Planar graphs
    • A graph is planar if it can be drawn in the plane without edge crossings.
    • For a connected planar graph, Euler's formula gives $V - E + F = 2$, where $F$ is the number of faces.
    • A simple planar graph satisfies $E \leq 3V - 6$.
  • Kuratowski's theorem
    • A finite graph is planar if and only if it contains no subdivision of $K_5$ or $K_{3,3}$.
