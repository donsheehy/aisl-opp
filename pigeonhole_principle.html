Pigeonhole Principle

Summary

The pigeonhole principle states that if more objects are distributed into boxes than there are boxes, some box must contain more than one object. Part of Combinatorics.

  • Simple form
    • If $n+1$ objects are placed into $n$ boxes, some box contains at least two objects.
    • Equivalently, no function $f : A \to B$ with $|A| > |B|$ is injective.
  • Generalized pigeonhole
    • If $N$ objects are placed in $k$ boxes, some box contains at least $\lceil N/k \rceil$ objects.
  • Ramsey-flavored applications
    • Existence proofs using the principle bypass explicit construction; for example, among any 6 people, there are 3 mutual acquaintances or 3 mutual strangers (a base case of Ramsey's theorem).
