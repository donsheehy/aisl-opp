Basic Counting

Summary

Basic counting comprises the fundamental rules for determining the sizes of finite sets. Part of Combinatorics.

  • Sum rule
    • If $A$ and $B$ are disjoint, then $|A \cup B| = |A| + |B|$.
  • Product rule
    • The number of ways to make a sequence of independent choices equals the product of the per-step counts: $|A \times B| = |A| \cdot |B|$.
  • Subtraction (complement) rule
    • For $A \subseteq U$, we have $|A| = |U| - |U \setminus A|$.
  • Division rule
    • If a counting procedure overcounts each object exactly $k$ times, divide the count by $k$.
    • Equivalently, if $f : A \to B$ is $k$-to-one then $|A| = k|B|$.
