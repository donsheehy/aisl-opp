Functions

Summary

A function $f : A \to B$ is a rule that assigns to each element $a \in A$ exactly one element $f(a) \in B$. Part of Functions and Relations.

  • Domain, codomain, image, preimage
    • The domain of $f$ is $A$ and the codomain is $B$.
    • The image of $f$ is $f(A) = {f(a) : a \in A} \subseteq B$.
    • The preimage of a set $S \subseteq B$ is $f^{-1}(S) = {a \in A : f(a) \in S}$.
  • Injective, surjective, bijective
    • $f$ is injective (one-to-one) if $f(a_1) = f(a_2)$ implies $a_1 = a_2$.
    • $f$ is surjective (onto) if $f(A) = B$.
    • $f$ is bijective if it is both injective and surjective.
  • Composition and inverse functions
    • The composition $(g \circ f)(a) = g(f(a))$ is defined when the codomain of $f$ equals the domain of $g$.
    • $f$ has a two-sided inverse if and only if $f$ is bijective.
  • pigeonhole_principle (as a function-counting tool)
    • If $|A| > |B|$, then no function $f : A \to B$ is injective.
