Set Operations

Summary

Set operations build new sets from existing ones using constructions such as union, intersection, and Cartesian product. Part of Set Theory. Builds on sets_basics.

  • Union, intersection, difference, complement
    • The union is $A \cup B = {x : x \in A \vee x \in B}$.
    • The intersection is $A \cap B = {x : x \in A \wedge x \in B}$.
    • The set difference is $A \setminus B = {x : x \in A \wedge x \notin B}$.
    • The complement of $A$ relative to a universe $U$ is $A^c = U \setminus A$.
  • Cartesian products
    • The Cartesian product is $A \times B = {(a,b) : a \in A, b \in B}$.
    • For finite sets, $|A \times B| = |A| \cdot |B|$.
  • Identities and Venn diagrams
    • De Morgan's laws state $(A \cup B)^c = A^c \cap B^c$ and $(A \cap B)^c = A^c \cup B^c$.
    • Distributivity states $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$.
