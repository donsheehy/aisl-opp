Graph Basics

Summary

A graph is a pair $G = (V, E)$ consisting of a set of vertices $V$ and a set of edges $E$, where each edge connects two vertices. Part of Graph Theory.

  • Degree and the handshake lemma
    • The degree $\deg(v)$ of a vertex $v$ is the number of edges incident to $v$ (loops count twice).
    • The sum of all vertex degrees equals twice the number of edges.
  • Common variants
    • A simple graph has no loops or parallel edges.
    • A multigraph allows parallel edges between the same pair of vertices.
    • A digraph (directed graph) has edges that are ordered pairs.
  • Substructure
    • $H$ is a subgraph of $G$ if $V(H) \subseteq V(G)$ and $E(H) \subseteq E(G)$.
    • Two graphs $G$ and $H$ are isomorphic, written $G \cong H$, if there is a bijection $V(G) \to V(H)$ preserving adjacency.

Handshake lemma: $\displaystyle\sum_{v \in V} \deg(v) = 2|E|$. In particular, the number of odd-degree vertices is even.
