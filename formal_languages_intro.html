Formal Languages (Intro)

Summary

Formal language theory is the mathematical study of strings and the sets of strings accepted by machines or generated by grammars. Part of Discrete Structures for Computation.

  • Alphabets, strings, languages
    • An alphabet $\Sigma$ is a finite set of symbols.
    • A string over $\Sigma$ is a finite sequence of symbols; $\Sigma^*$ denotes all such strings including the empty string.
    • A language over $\Sigma$ is any subset $L \subseteq \Sigma^*$.
  • Context-free grammars
    • A context-free grammar (CFG) is a tuple $(V, \Sigma, R, S)$ where $V$ is a set of variables, $\Sigma$ is the terminal alphabet, $R$ is a set of rules of the form $A \to \alpha$ with $A \in V$ and $\alpha \in (V \cup \Sigma)^*$, and $S \in V$ is the start symbol.
    • The language $L(G)$ is the set of terminal strings derivable from $S$.
    • Context-free languages strictly contain the regular languages of finite_state_machines; for example, ${a^n b^n : n \geq 0}$ is context-free but not regular.
