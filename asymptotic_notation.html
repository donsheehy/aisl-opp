Asymptotic Notation

Summary

Asymptotic notation provides a vocabulary for comparing the growth rates of functions or sequences, suppressing constant factors and lower-order terms. Part of Sequences, Asymptotics, and Discrete Calculus.

  • Big-O, big-Omega, big-Theta
    • We write $f = O(g)$ (read "big-O of $g$") if there exist $c, n_0$ such that $f(n) \leq c g(n)$ for all $n \geq n_0$.
    • We write $f = \Omega(g)$ (read "big-Omega of $g$") if there exist $c, n_0$ such that $f(n) \geq c g(n)$ for all $n \geq n_0$.
    • We write $f = \Theta(g)$ (read "big-Theta of $g$") if both $f = O(g)$ and $f = \Omega(g)$ hold.
  • Little-o, little-omega
    • We write $f = o(g)$ if $\lim_{n \to \infty} f(n)/g(n) = 0$, meaning $f$ grows strictly slower than $g$.
    • We write $f = \omega(g)$ if $\lim_{n \to \infty} f(n)/g(n) = \infty$, meaning $f$ grows strictly faster than $g$.
  • Common growth rates
    • The standard hierarchy is $1 \prec \log n \prec \sqrt{n} \prec n \prec n \log n \prec n^2 \prec n^k \prec 2^n \prec n!$.
