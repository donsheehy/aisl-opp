Set Cardinality

Summary

The cardinality of a set is a measure of its size, generalizing counting to infinite sets via bijections. Part of Set Theory.

  • Finite vs. infinite sets
    • A set $A$ is finite if there is a bijection $A \to {1, \ldots, n}$ for some $n \geq 0$.
    • Two sets have the same cardinality, written $|A| = |B|$, if and only if there exists a bijection $A \to B$.
  • Countable and uncountable sets
    • A set $A$ is countable if it is finite or has a bijection with $\mathbb{N}$; otherwise it is uncountable.
    • The sets $\mathbb{Z}$ and $\mathbb{Q}$ are countable; $\mathbb{R}$ is uncountable.
  • Cantor's theorem and diagonalization
    • Cantor's theorem states $|A| < |\mathcal{P}(A)|$ for every set $A$.
    • Cantor's diagonal argument shows that $\mathbb{R}$ is uncountable.

Closely related to functions (bijections) and to the structure of $\mathbb{N}$.
