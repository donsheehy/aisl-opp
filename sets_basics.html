Set Basics

Summary

A set is an unordered collection of distinct elements, taken as a primitive notion in mathematics. Part of Set Theory.

  • Membership, subsets, equality, power sets
    • We write $x \in A$ to say that $x$ is a member (or element) of $A$.
    • $A$ is a subset of $B$, written $A \subseteq B$, if $\forall x (x \in A \rightarrow x \in B)$.
    • Two sets are equal ($A = B$) if and only if $A \subseteq B$ and $B \subseteq A$ (the axiom of extensionality).
    • The power set of $A$ is $\mathcal{P}(A) = {S : S \subseteq A}$; for finite $A$, $|\mathcal{P}(A)| = 2^{|A|}$.
  • Set-builder notation
    • The notation ${x \in U : P(x)}$ specifies the set of all elements of $U$ satisfying predicate $P$.
