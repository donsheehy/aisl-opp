Probabilistic Method (Intro)

Summary

The probabilistic method is a non-constructive proof technique, pioneered by Erdős, that establishes the existence of a combinatorial object by showing it occurs with positive probability under a suitable distribution. Part of Discrete Probability.

  • Existence proofs via probability
    • To show some object with property $P$ exists, sample uniformly from a class and prove $P$ holds with positive probability.
    • If $\mathbb{E}[X] \ge c$, then there must exist an outcome in which $X \ge c$.
  • Random graphs
    • The Erdős–Rényi model $G(n, p)$ includes each edge independently with probability $p$.
    • Threshold phenomena occur when graph properties (connectivity, giant component) appear sharply at a critical $p(n)$.
    • Erdős's lower bound on Ramsey numbers via $G(n, 1/2)$ is the classical first application of the method.
