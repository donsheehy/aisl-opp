Graph Representations

Summary

Graph representations are data structures for storing a graph in memory, with different trade-offs between space, query time, and traversal efficiency. Part of Graph Theory.

  • Adjacency matrix
    • An adjacency matrix is an $n \times n$ matrix $A$ with $A_{ij} = 1$ if and only if ${i,j} \in E$ (or edge multiplicity for multigraphs).
    • It uses $\Theta(n^2)$ space, supports $O(1)$ adjacency queries, and satisfies: $A^k_{ij}$ counts walks of length $k$.
  • Adjacency list
    • An adjacency list stores, for each vertex, the list of its neighbors.
    • It uses $\Theta(n + m)$ space and is efficient for sparse graphs and traversal algorithms.
  • Incidence matrix
    • An incidence matrix is an $n \times m$ matrix $B$ with $B_{ve} = 1$ if and only if vertex $v$ is incident to edge $e$.
    • It is useful in algebraic graph theory and for signed or oriented variants.
